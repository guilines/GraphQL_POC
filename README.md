# GraphQL POC

## Getting Started
This repository is ready to be split in 3 modules:
- frontend
- backend
- schemas

### Frontend
#### Init
```
cd frontend
npm install
npm run codegen
```

#### Launching the frontend app:
```
npm start
```
#### Launching the storybook:
```
npm run storybook
```
### Backed
#### Init
```
cd backend
npm install
npm run generate
npm run start
```
